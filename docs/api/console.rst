边玩边探索
==============================

..  module:: wxpy

想要做点小实验，调试代码，或是探索 wxpy 的功能特性？反复修改和运行确实太麻烦。

试试下面两种"边玩边探索"的方式，告别涂涂改改。


在代码中进行探索
------------------------------

..  note:: 这种方式适合在现有的代码中进行探索。

只需将 :any:`embed()` 放在任何你想探索的位置。运行后，你就可以从这个位置开始探索了。

例如，初始化一个机器人，然后看看它能做些什么::

    from wxpy import *
    robot = Robot()
    embed() # 开始探索

    >>> robot
    # Out[1]: <Robot: 游否>
    >>> robot.friends()
    # Out[2]: [<Friend: 路人甲>, <Friend: 路人乙>, <Friend: 路人丙>]

..  autofunction:: embed
